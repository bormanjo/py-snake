# Py-Snake

> A python implementation of the video game classic Snake via Python's Pygame library

## Overview

## How to Play

### As a Human Player

- start the game:

```{bash}
python -m game
```

- Use the `Up`, `Down`, `Left`, `Right` arrow-keys to control the snake

### As an AI Player

- start the game:

```{bash}
python -m game --ai_player
```
